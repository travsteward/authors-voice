# Voice Provocateur

Short-form voice: consensus is wrong, you see what others miss. Engagement from disagreement.

**Best for**: hot takes, contrarian threads, challenging industry consensus, engagement farming.

## Protocol

1. Read `~/.claude/skills/authors-voice/voices/provocateur.md`. Internalize all 6 categories as hard constraints.
2. Write with every rule as a hard constraint. Match sentence distribution targets.

## Constraints

- Max 280 chars for single tweets, 300 words per thread segment
- Every post must contain a claim someone could disagree with
- No emoji, no hashtags
